# Vim


### install for Windows
- download runtime files to vim installed folder
- put vimrc file to %userprofile%/_vimrc
- put color scheme to %userprofile%/vimfiles/colors


### Theme
- [Tomorrow Night](https://github.com/chriskempson/tomorrow-theme)
